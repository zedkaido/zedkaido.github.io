XES (X Engine Selector)

Why would you use duck duck go !bangs when you can have a faster, local, private, open-source, configurable solution.

That's the idea behind XES aka. X Engine Selector.

If you've been living under a rock, !bangs are shortcuts to other search engines. For example, you can quickly search wikipedia with !wiki Hutter Prize, Google with !g do a barrell role, or YouTube with Google IO 2025 !yt.

If you are happy with xes.zedkaido.com bangs, just set https://xes.zedkaido.com/?q=%s as your default search engine.

XES's JS is cached locally, so redirects are done in your browser.

Otherwise, you can clone github.com/zedkaido/xes and edit src/bangs.js bangs to your liking. It's quite easy.

src/bangs.js 
export const bangs = {
	"g": {
		name: "Google Search",
		url: "https://www.google.com/search?q={{{s}}}"
	},
	// ...
}
